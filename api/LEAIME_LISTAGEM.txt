Para que a listagem de empresas, cursos e vagas não de erro quando for rodar a aplicação no servidor local.

Passo1: Execute os seguintes passsos:
	Digite no terminal:
				python manage.py makemigrations api
				python manage.py sqlmigrate api 0001
				python manage.py migrate

Os comandos cima servem para criar as tabelas do banco.
Para popular o banco de dados, abra um editor bd e execute os comandos presentes no arquivo api/Busca/comandos_sql.txt num editor de BD e salve as insercoes feitas.

Para rodar entre na pasta e inicialize com o comando: 	python manage.py runserver

para visualizar a listagem geral das empresas:		digite no navegador		127.0.0.1:8000/api/busca/SearchCompany/

para visualizar a listagem geral das vagas:			digite no navegador		127.0.0.1:8000/api/busca/SearchVacancy/

para visualizar a listagem geral dos cursos:		digite no navegador		127.0.0.1:8000/api/busca/course/

para visualizar a Busca das empresas que oferecem vagas de acordo com o curso do aluno: 	
													digite no navegador		127.0.0.1:8000/api/busca/SearchCompany/?id=123

para visualizar a Busca por uma empresa especifica: 	
								digite no navegador por exemplo:			127.0.0.1:8000/api/busca/SearchCompany/?name=Microsoft

para visualizar a Busca das vagas com relação ao curso do aluno(usuario), onde o parametro passado é o id_user: 	
								digite no navegador	por exemplo:			127.0.0.1:8000/api/busca/SearchOportunity/?id=123

Para buscar vagas de acordo com o curso, nome da empresa ou data de inserção:

Digite no navegador: 127.0.0.1:8000/api/busca/SearchOportunity/?course=&company=&icreated=&fcreated=

Onde, na query string,
    course recebe a abreviação do curso
    company recebe o nome da empresa
    icreated recebe a data inicial de filtragem (no formato YYYYMM)
    fcreated recebe a data final de filtragem (no formato YYYYMM)

A ordem que os parametros sao passados na query string nao importa
Alguns exemplos,
    127.0.0.1:8000/api/busca/SearchOportunity/?course=ADM
    - mostra todas as vagas do curso cuja abreviacao eh ADM
    - o filtro nao diferencia maiusculas de minusculas, tanto ADM quanto devem adm funcionar igualmente

    127.0.0.1:8000/api/busca/SearchOportunity/?company=Exemplo
    - mostras todas as vagas ofertadas pela empresa Exemplo
    - assim como acima, nao diferencia maiusculas e minusculas

    127.0.0.1:8000/api/busca/SearchOportunity/?icreated=201705
    - mostra todas as vagas criadas desde maio de 2017 (incluindo maio)
    - o formato deve ser YYYYMM

    127.0.0.1:8000/api/busca/SearchOportunity/?fcreated=201705
    - mostra todas as vagas criadas antes de maio de 2017 (incluindo maio)
    - o formato deve ser YYYYMM

    127.0.0.1:8000/api/busca/SearchOportunity/?fcreated=201705&icreated=201701
    - mostra todas as vagas criadas entre janeiro e maio de 2017 (incluindo janeiro e maio)

    127.0.0.1:8000/api/busca/SearchOportunity/?course=ADM&company=Exemplo
    -mostra todas as vagas ofertadas pela empresa Exemplo para o curso cuja abreviação é ADM
